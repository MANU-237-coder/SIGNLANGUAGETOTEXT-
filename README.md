# ✋ Sign Language to Text Converter

## 📌 Overview

This project is a **Sign Language to Text Converter** that uses machine learning to recognize hand gestures and convert them into readable text in real-time.

It helps bridge communication between **hearing-impaired individuals** and others.

---

## 🚀 Features

* 📷 Real-time hand gesture detection using camera
* 🤖 Machine Learning model for prediction
* 🔤 Converts sign language into text
* 🌐 Frontend interface for user interaction
* ⚡ Fast and responsive system

## ⚙️ Installation & Setup

### 🔹 1. Clone the repository

```
git clone https://github.com/MANU-237-coder/SIGNLANGUAGETOTEXT-.git
cd SIGNLANGUAGETOTEXT
```

---

### 🔹 2. Backend Setup

```
cd backend
pip install -r requirements.txt
python app.py
```

---

### 🔹 3. Frontend Setup

```
cd frontend
npm install
npm start
```

## 📸 How It Works

1. Camera captures hand gestures
2. Image is preprocessed
3. Model predicts the sign
4. Output is converted into text
